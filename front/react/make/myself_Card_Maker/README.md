# 내가 만든 카드 웹 사이트 


---


## 깃 허브 페이지 https://sksn12.github.io/myself_Card_Maker/#/


### 문제점 (해결 파이어베이스에 호스팅 권한 설정)
_리액트 라우팅을 사용하고 동적인 페이지를 지원하는 깃허브 페이지에 업로드하면 제대로 사용이 되지않아 hashRouter를 사용하였지만 여전히 로그인 팝업창이 나오자마자 닫힘(원인 모름)_
_로컬 호스트에서는 제대로 작동됨_

---


## 로그인 되엇을시에 리다이렉션 후 밑에 이미지 페이지가 나와 카드를 제작 할 수 있어야함
